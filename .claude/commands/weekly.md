# Weekly Autonomous Review Cycle

Execute the complete weekly review and maintenance workflow:

## Full Cycle (Minimal User Intervention)

1. **Performance Review** (`review` skill)
   - Analyze all closed trades this week
   - Calculate win rate, avg return, P&L
   - Extract lessons learned
   - Update patterns.md with new patterns

2. **Deep Catalyst Scan** (`scan` skill)
   - Comprehensive scan of all sources
   - FDA PDUFA calendar (next 90 days)
   - SEC 13D filings (last 7 days)
   - Merger/spin-off announcements
   - Legislative developments
   - Goal: Maintain 8-12 active catalyst events

3. **Watchlist Maintenance**
   - Check for stale watchlist items (>7 days old)
   - Re-score if material news or price changes
   - Archive outdated events

4. **Precedent Index Update**
   - Verify precedents/index.json is current
   - Add tags from this week's closed trades
   - Update patterns.md

5. **Framework Calibration Check**
   - Review kill screen effectiveness
   - Check if scoring thresholds need adjustment
   - Flag any systematic issues

6. **Weekly Summary**
   - Week's performance metrics
   - New catalysts added
   - Active positions status
   - Regime changes
   - Action items for next week

## Token-Efficient Design

Weekly deep-dive complements daily incremental work:
- Daily: Process 2-3 new events
- Weekly: Catch anything missed, deep analysis, learning

## When to Run

Friday after market close OR Sunday evening before the week starts.
